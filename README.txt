# Expense Tracker

A simple Expense Tracker application built using Python's Tkinter for the GUI and Matplotlib for visualizing expense data.

## Features

- Add expenses with amount, category, and date.
- View a pie chart representing the distribution of expenses by category.

## Requirements

- Python 3.x
- Tkinter (comes pre-installed with Python)
- Matplotlib (install via pip)
